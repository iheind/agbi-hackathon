### Installation ğŸ“¦:
* ## Digital Marketing Webapp
* Clone the repo
```sh
   $ git clone https://github.com/Sai-Adarsh/agbi-hackathon
```
```sh
   $ cd agbi-hackathon/diagnosis-tagging
```
* Create virtualenv
```sh
   $ virtualenv venv
```
* For Linux
```sh
   $ source ./venv/bin/activate
```
* For windows
```sh
   $ cd venv/Scripts/
```
```sh
   $ activate
```
* Install dependencies
```sh
   $ pip install -r requirements.txt
```
